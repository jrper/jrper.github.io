Trying to keep some living notes on how markdown works. 

#### Headers


{% highlight markdown %}

# This is an H1 #

## This is an H2 ##

###### This is an H6 ######

{% endhighlight %}


#### Links 


{% highlight markdown %}

This is [an inline link](http://bob.com)

This is [a reference link][id]

[id] http://bob.com "This is a website title."

{% endhighlight %}

#### Emphasis

{% highlight markdown %}

*single asterisks*

_single underscores_

**double asterisks**

__double underscores__

{% endhighlight %}
